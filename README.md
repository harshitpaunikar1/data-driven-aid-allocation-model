# Data-Driven Aid Allocation Model

> **Domain:** Financial Analysis

## Overview

Transparent allocation framework for NGOs using multi-criteria analysis, principal component analysis, and linear programming under budget constraints with sensitivity testing.

## Skills & Technologies

- Data Engineering
- Indicator Normalization
- Multi-Criteria Decision Analysis
- Linear Programming
- Clustering
